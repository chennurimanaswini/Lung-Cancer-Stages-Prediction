# Lung Cancer Stages Prediction using Machine Learning

This project predicts the stage of lung cancer using machine learning and deep learning techniques based on medical imaging and clinical data.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Scikit-learn
- CNN / ANN models

## Project Description
The system analyzes medical images and patient data to classify lung cancer into different stages. This helps doctors and healthcare professionals in early detection and treatment planning.

## Features
- Image preprocessing
- Deep learning model training
- Lung cancer stage classification
- Prediction system

## How to Run the Project

1. Install required libraries

pip install -r requirements.txt

2. Run the model

python main.py
